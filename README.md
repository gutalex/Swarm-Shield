# Swarm-Shield 🛡️🛸

An open-source distributed air defense simulation framework against mass swarm drone attacks.

## Current Phase: R&D MVP (Simulation)

Testing target allocation algorithms (Hungarian Linear Assignment, Kalman Filter prediction) with real-time 3D tracking visualizations.

### Quick Start with Python:
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run simulation:
   ```bash
   python sim/main_simulation.py
   ```
